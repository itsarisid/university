# Focus On Development Without Coding
Build and launch full-fledged web applications with IIWI’s no-code platform.

### Overview

- ##### Code-less
    IIWI Editor quickly generates custom applications. No need for development.

- ##### Developer Friendly
    Unlock the developer mode to get further details on technicalities.

- ##### Scalable
    No-Code Studio comes with massive room for scalability. A perfect choice for small growing enterprises and big enterprises alike!

### Screen Designer

- ##### Forms
    Feature-rich No-Code Studio creates multiple forms like notebooks, integrated statistics, and more!

- ##### Lists
    IIWI No-Code Studio sheets work just like Excel Spreadsheets. Comes with two types of lists - read-only and read-write.

- ##### Calendar
    Keeps you on track with your schedule! Documents are linked with your schedule.

### Features

- ##### Automations
    No-Code Studio

- ##### Menu Editor
    Easy to organize and edit your menu. Create new menus or edit with easy and swift ‘drag’ and ‘drop’.

- ##### Search
    Categorizes your custom search bar with advanced features like access rights, filters, and groups.

- ##### Conditioned Properties
    Keep your fields invisible with IIWI Editor. The fields can be read under specific conditions only.

### Reports

- ##### XML Editor
    Gives access to advanced reporting customizations in HTML format.

- ##### Templates
    This allows you to generate your reports with ease. Gives access to multiple report templates.

### Automation

- ##### Send Emails
    Generates automated emails dependent on actions, conditional statements, or time. Keeps the customer satisfied and in the loop all the time!

- ##### Python Code
    Create any numbers and types of advanced automation by using the Python code.

- ##### Update Values
    Takes care of automated updates based on predefined conditional statements.

### Miscellaneous

- ##### Inherit IIWI Features
    Benefit from existing IIWI features to create advanced applications in no time!

- ##### Access Rights
    Make filters or ACLs rules to define access rights for bigger companies.

- ##### Icon Designer
    Icon Designer allows you to design yourself. You can create a creative logo in no time.
